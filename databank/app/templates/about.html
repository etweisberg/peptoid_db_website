The Peptoid Data Bank

This is a techical demonstration of my implementation of a web development using Flask to create a searchable database for published peptoid structures.

Development

Development of this technical demonstration was done by Ethan Weisberg using Flask, a web framework for Python. Bootstrap was used as the framework for CSS, which is used in the layout of the website. Flask-sqlalchemy was used for the backend database. For more information see: GitHub
