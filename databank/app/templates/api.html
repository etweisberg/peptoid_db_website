PeptoidDB API

Peptoid Data Bank - Our API

Our API allows peptoid researchers to retrieve data directly from our database in their own programs and scripts. Researchers are encouraged to use this tool for chemoinformatic data collection, but should be advised that requests are limitted to 1 per minute. If REST API's aren't your cup of tea and you don't want to follow our predefined routes, we also offer a GraphQL API, which you can explore by clicking below or at the route /graphql.

GraphQL API

Routes

'api/peptoids/{CODE}'
Returns a JSON object of a peptoid's properties
'api/peptoids/{CODE}/residues'
Returns a JSON object for a peptoid's residues
'api/peptoids/{CODE}/authors'
Returns a JSON object for a peptoid's authors
'api/peptoids'
Returns a JSON object of URLs for all peptoids in the database
